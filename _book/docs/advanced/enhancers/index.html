Enhancers

Enhancers

In standard Redux, enhancers can be used to enhance the store with third-party capabilities. Many of the existing Redux enhancers will just work with Redux Subspace, but enhancers that try to use the store's dispatch function can cause some problems for namespaced subspaces as it may bypass the subspace and go directly to the root store.

Fixing this generally involves creating some form of wrapper around the thing the enhancer trying to use to augment it's functionality to be aware of subspaces.
